> 版本：v2（参数专项优化版）
> 回测区间：2023-04-17 ～ 2026-04-15
> 标的：A 股主流宽基 ETF × 6
> 成本假设：单边手续费 0.15%，无印花税

---

## 一、策略总览

非对称 DTS（Directional Trend Signal）是一套**双时间框架趋势跟踪系统**，由**周线许可层（WTS）**和**日线滤波层（DTS）**两个模块组成，策略行为对多空**非对称**：

- **多头信号（进场）**：由 WTS 周级别趋势主导，确认大方向做多
- **空头信号（出场）**：由 DTS 日级别信号即时响应，灵活保护利润

核心思想：**大框架定方向，小框架管风险**。

---

## 二、核心模块：七规则趋势系统

WTS 和 DTS 使用同一套「七规则」（7-Rules）框架，差异仅在时间周期和参数阈值。

### 2.1 K 线分类

对任意一根 K 线，先计算振幅：

```
振幅 = (最高价 - 最低价) / 开盘价
```

然后按阈值 `FLAT_THR` 分类：

| 条件 | 分类 |
|------|------|
| 振幅 < FLAT_THR | `FLAT`（平缓，无方向） |
| 振幅 ≥ FLAT_THR 且收盘 ≥ 开盘 | `UP`（上涨） |
| 振幅 ≥ FLAT_THR 且收盘 < 开盘 | `DOWN`（下跌） |

周线参数：`FLAT_THR = 1.5%`
日线参数：`FLAT_THR = 0.5%`

### 2.2 七规则判定

对相邻两根 K 线（前根 `prev`，当根 `curr`），根据组合类型执行以下规则：

| 规则 | prev 方向 | curr 方向 | 幅度滤波器 | 输出信号 |
|------|-----------|-----------|-----------|---------|
| Rule 1 | UP | UP | SAME 门槛 | +1（做多）|
| Rule 2 | DOWN | DOWN | SAME 门槛 | -1（做空）|
| Rule 3 | UP | DOWN | REV 门槛 | -1（做空）|
| Rule 4 | DOWN | UP | REV 门槛 | +1（做多）|
| Rule 5 | FLAT/UP | UP/FLAT | 无 | +1（直接做多）|
| Rule 6 | FLAT/DOWN | DOWN/FLAT | 无 | 0（直接做空）|
| Rule 7 | FLAT | FLAT | 无 | 保持原仓 |

**幅度滤波逻辑（关键防噪设计）**：

Rule 1～4 触发时，需要额外比较两根 K 线的振幅差：

```
幅度差 = |curr_振幅 - prev_振幅|

若方向相同（Rule 1/2）：需要 幅度差 ≥ SAME 阈值 才改变仓位
若方向相反（Rule 3/4）：需要 幅度差 ≥ REV 阈值 才改变仓位
若幅度差不足：维持原仓，不改变
```

**设计意图**：过滤掉振幅细微的假突破，只响应有力度的趋势信号。

### 2.3 参数含义

| 参数 | 含义 | 周线固定值 | 日线优化范围 |
|------|------|-----------|------------|
| `SAME` | 同向延续的最小幅度差门槛 | 0.90% | 0.30 ～ 5.00 |
| `REV` | 逆向反转的最小幅度差门槛 | 0.45% | 0.15 ～ 2.50 |

## 三、WTS 周线许可层

### 3.1 作用

WTS 提供「本周是否允许持多」的二值信号（0 或 1），作为日线层的**上级许可**。
WTS = 1 → 本周多头模式开启
WTS = 0 → 本周无论日线如何，强制空仓

### 3.2 计算方式

1. 将日线数据聚合为周 K 线（周一开盘 → 周五收盘）
2. 对周 K 线序列运行七规则，得到每周的周线仓位 `wts`
3. **信号错位一格**：本周计算出的 `wts`，用于指导**下一周**的持仓许可

```
本周 K 线 → 七规则 → wts_当周 → 下周许可信号
```

### 3.3 固定参数

周线参数为全局统一值，不做 ETF 级别优化：

```
W_FLAT = 1.5%
W_SAME = 0.90%
W_REV  = 0.45%
```

## 四、DTS 日线滤波层

### 4.1 作用

在 WTS = 1（允许持多）的周内，DTS 进一步判断每日是否实际持有多头仓位。

核心逻辑：进场容易，出场灵敏——
- **新多头周开始**：只要 WTS 翻 1，周一直接入场（v2 原版）
- **多头周内出场**：只要 DTS 信号 = 0，当日立即平仓
- **多头周内再入场**：DTS 信号恢复 = 1，当日再度买入

### 4.2 信号计算

1. 对日 K 线运行七规则，得到每日仓位 `dts`
2. **信号错位一天**：`dts_s = dts.shift(1)`（昨天信号，今天执行）

```
昨日 K 线 → 七规则 → dts_昨天 → 今天执行 dts_s
```

### 4.3 持仓状态机（核心逻辑）

```python
for 每个交易日:
    wp  = WTS许可（本周）
    dts = dts_s（昨日DTS信号，今日使用）

    if wp == 0:
        pos = 0          # WTS 不许可 → 强制空仓

    elif 前一日wp==0 且 今日wp==1:
        pos = 1          # 新多头周开始 → 周一直接入场

    else:                # 多头周内
        if pos == 1 and dts == 0:
            pos = 0      # DTS 转空 → 出场
        elif pos == 0 and dts == 1:
            pos = 1      # DTS 转多 → 再入场
        # 其他情况：维持现有仓位
```

**"非对称"的含义**：
- 入场条件宽松（WTS 翻多 → 直接做多）
- 出场条件严格（DTS 即时响应）
- 多头持续跟随，空头快速离场

## 五、v1 → v2 参数优化

### 5.1 问题发现

v1 的日线参数来自**对称 DTS 策略**的优化结果。但非对称策略的出场机制完全不同：

- 对称策略：DTS=0 和 DTS=1 均需要过滤，SAME/REV 均重要
- 非对称策略：出场依赖 `DTS=0`，REV 阈值不够高 → **反转噪声过多** → 频繁出场再入场

典型问题：「1 日交易」——周一入场，周二 dts_s=0，立即出场，双向手续费白交。

### 5.2 优化方法

针对非对称策略，对全 3 年数据进行网格搜索：

```
SAME 搜索空间：[0.30, 0.50, 0.75, 1.00, 1.25, 1.50, 2.00, 2.50, 3.00, 3.50, 4.00, 5.00]
REV  搜索空间：[0.15, 0.30, 0.45, 0.60, 0.75, 1.00, 1.25, 1.50, 2.00, 2.50]
优化目标：最大化超额 Alpha（策略收益 - BNH 收益）
最少交易笔数：3 笔（防止过拟合极端参数）
```

### 5.3 优化结果

| 标的 | v1 SAME/REV | v2 SAME/REV | Alpha 改善 |
|------|-----------|-----------|-----------|
| 上证50ETF | 0.75 / 0.60 | **1.50 / 2.50** | +7.9% |
| 沪深300ETF | 2.50 / 2.00 | **2.50 / 2.50** | +2.8% |
| 中证500ETF | 2.50 / 2.00 | **2.50 / 2.50** | +21.4% |
| 科创50ETF | 2.50 / 1.50 | 2.50 / 1.50（不变）| — |
| 深证100ETF | 3.00 / 2.00 | 3.00 / 2.00（不变）| — |
| 创业板ETF | 4.00 / 1.00 | 4.00 / 1.00（不变）| — |

**规律**：需要提高 REV 阈值的标的，均是因为日线反转噪声较多（振幅小但方向频繁切换）。

## 六、交易执行细节

| 项目 | 规则 |
|------|------|
| 入场价格 | 周一（或再入场日）**开盘价**买入 |
| 出场价格 | 信号翻空次日**开盘价**卖出 |
| 手续费 | 单边 0.15%（买入 +0.15%，卖出 -0.15%） |
| 实际买入成本 | `开盘价 × (1 + 0.15%)` |
| 实际卖出所得 | `开盘价 × (1 - 0.15%)` |
| 仓位 | 全仓（0 或 100%），无杠杆 |
| 空头处理 | 空仓等待，不做空 |

## 九、策略的核心优势

1. **熊市主动保本**：通过日线 DTS 信号感知趋势衰竭，及时退出，大幅降低熊市回撤
2. **牛市全程持仓**：在趋势明确的多头周内，持仓不受干扰，充分捕获上涨
3. **复利效应放大**：熊市少亏的本金以更大基数进入牛市，同样的涨幅带来更大绝对收益
4. **参数有物理意义**：SAME/REV 门槛对应真实的「趋势延续力度」和「反转强度」，可解释、可调整

---

## 十、局限性

1. **急涨行情跑输 BNH**：创业板 ETF 在 2024-2025 年几次集中式暴涨中，策略因信号错位 1 天而缺席，无法弥补
2. **参数样本内优化**：v2 参数在 3 年全样本上优化，存在一定过拟合风险，建议结合走前验证（Walk-Forward）
3. **最低手续费敏感**：若手续费高于 0.3% 单边，高频切换 ETF 的策略效果会显著下降
4. **流动性假设**：策略假设以开盘价全仓买卖，大资金规模需考虑冲击成本
