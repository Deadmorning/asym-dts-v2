## 项目概述

本项目实现并验证了**非对称 DTS v2 策略**——一套基于周线趋势许可（WTS）+ 日线趋势滤波（DTS）的量化交易策略。

核心特点：
- **非对称**：进场由周线主导（慢），出场由日线即时响应（快）
- **双层过滤**：周线大方向 + 日线细节，减少噪声交易
- **v2 参数**：专为非对称出场机制重新优化 DTS 参数，显著改善 Alpha

## 策略逻辑简述

完整说明见 [docs/strategy.md](docs/strategy.md)

### 双层框架

```
周线层（WTS）
  └─ 周 K 线七规则 → 本周趋势方向
     └─ WTS=1：下周允许持多
     └─ WTS=0：下周强制空仓

日线层（DTS）
  └─ 日 K 线七规则 → 当日趋势状态
     └─ 新多头周 → 周一直接入场
     └─ DTS=0 → 次日开盘出场（非对称！）
     └─ DTS=1 → 次日开盘再入场
```

### 非对称的含义

| 方向 | 信号来源 | 响应速度 | 目的 |
|------|---------|---------|------|
| 做多（入场） | WTS 周线 | 慢（周级别） | 跟随大趋势 |
| 出场（空仓） | DTS 日线 | 快（日级别） | 即时保护利润 |

### 七规则幅度过滤

```python
幅度差 = |当根振幅 - 前根振幅|

同向(Rule 1/2): 需 幅度差 ≥ SAME 才改变仓位
反向(Rule 3/4): 需 幅度差 ≥ REV  才改变仓位
否则：维持原仓
```

## v2 参数优化结果

| 标的 | v1 SAME/REV | v2 SAME/REV | Alpha 改善 |
|------|------------|------------|-----------|
| 上证50ETF | 0.75/0.60 | **1.50/2.50** | +7.9% |
| 中证500ETF | 2.50/2.00 | **2.50/2.50** | +21.4% |
| 科创50/深证100/创业板 | 原值 | 不变 | — |

优化逻辑：非对称策略的出场依赖 `DTS=0`，需要更高的 `REV` 门槛来过滤反转噪声，避免弱势日被频繁拉出仓位。
